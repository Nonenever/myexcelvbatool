# Excel-Formula-To-Text-Tool
Excel VBA 公式转文本说明工具

## 📖 项目简介
基于VBA自定义函数开发，可自动统计行内可见单元格数量，同时将单元格公式解析转换为可读文本表达式，自动剔除ROUND函数、提取标识名称，快速生成计算备注说明，适用于工程结算、财务报表、数据台账场景。

## ✨ 核心功能
1. 统计当前行左侧未隐藏列的单元格数量，隐藏列自动忽略不计
2. 自动解析单元格公式，将单元格地址替换为自定义名称文本
3. 批量清除公式内ROUND函数，保留原始计算逻辑
4. 识别`名称=数值`格式，精准截取等号前标识文字
5. 拼接计数结果与表达式，一键生成完整说明文本

## 🛠️ 安装使用
### 1. 导入代码
1. 打开Excel表格，按下 `Alt + F11` 调出VBA编辑器
2. 右键工程资源管理器，选择「插入」-「模块」
3. 将项目内VBA完整代码粘贴至编辑窗口
4. 文件另存为 **.xlsm** 启用宏工作簿格式
