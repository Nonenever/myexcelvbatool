# 合同支付申请单 VBA 工具 - 使用说明

## 1. 如何导入代码
1. 打开 Excel
2. 按下 `Alt + F11` 打开 VBA 编辑器
3. 右键工作表 → 插入 → 模块
4. 将 `.vba` 代码全部粘贴进去
5. 保存文件为 `.xlsm` 启用宏格式

## 2. 如何运行
1. 返回 Excel 界面
2. 选中一个空白单元格（作为表格左上角起点）
3. 按下 `Alt + F8`
4. 选择 `合同支付申请单_统一变量版`
5. 执行，自动生成完整表格

## 3. 如何修改内容（统一变量，最方便）
所有文字、数据、编号、账户信息都在代码最上方：
